# Void Launcher

Void Launcher is a modern Minecraft launcher built with Electron and React.

## Features

- Create and manage multiple Minecraft instances
- Support for Vanilla, Fabric, Quilt, Forge, and NeoForge
- Built-in Modrinth browsing for mods, modpacks, resource packs, shader packs, and data packs
- Offline and Microsoft account support
- Skin management for official and offline accounts
- Java detection, Java path selection, and managed Java downloads
- Backup and restore tools for instances
- Console viewer, health checks, and launcher diagnostics
- GitHub release update checks


## Installation

1. Go to the GitHub `Releases` page for this repository.
2. Download the latest Windows installer or portable build.
3. Run the installer and complete setup.


## Updating the launcher

Void Launcher can check GitHub Releases and prompt users when a newer version is available.
